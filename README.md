# Setting Up

The following notebook and utilities were created with python 3.8.5 and the following module versions:
- numpy 1.19.2
- pandas 1.1.3
- plotly 4.14.1
- geopy 2.1.0
- sklearn 0.23.2

To run the notebook, the training and test datasets should be saved in "eiq_ds_data/eiq_data.csv" and 
"eiq_ds_data/eiq_test.csv" respectively.
